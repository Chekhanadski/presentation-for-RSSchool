ORIGINS OF

WWW/HTML/CSS and AJAX/SPA

WORLD WIDE WEB

In 1989, Tim Berners-Lee invented the World Wide Web (WWW).

EXAMPLE

The first website ever was info.cern.ch, which had information about the World Wide Web Project. It came out in 1991. This was the first time people used the WWW.

HTML & CSS

HTML was made to structure web pages using tags. But they looked pretty basic until CSS came along in 1996. With CSS, developers could make web pages look better by adding colors, fronts, and layouts.

FACT ABOUT AJAX

The term AJAX was first publicly used in 2005 by Jesse James Garrett. However, in one form or another, many of these technologies were available and used as early as 1996.

SINGLE PAGE APPLICATION

Instead of loading a new page from the server every time you click on something, SPAs load once and then update the content dynamically as needed.

CONCLUSION

HTML, CSS, Ajax, and SPA technologies have shaped our usage of the internet today, making it more interactive and user-friendly. Understanding this history helps us appreciate the incredible progress we have made and inspires us to continue pushing the boundaries.
